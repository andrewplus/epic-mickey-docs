Riivolution Patch Help

<- back

This tool allows for custom cmdline.txt files to be generated for Epic Mickey to load levels from any point in the game. The "Download Riivolution Patch" feature allows for these files to be used on a softmodded Wii console using Riivolution, an on-the-fly game patcher that allows users to place content on SD cards to be read by the game during regular play. This app is typically used for game modding.

You will need...

  • A softmodded Wii (guide)
  • Riivolution
  • An SD Card
  • Epic Mickey Wii disc (US version tested, others untested but should work)

Guide

  1. Insert your Wii's SD card into your computer. Make sure you have Riivolution installed.
  2. Go to the generator and select a level. Once you're done, click the "Download Riivolution Patch" button. A ZIP file should begin downloading.
  3. Extract the ZIP file that you downloaded to the root of your SD card. If you're asked if you want to replace files or merge folders, click OK to all.
  4. Remove the SD card from your computer and insert it into your Wii or Wii U console. Make sure your Epic Mickey game disc is inserted.
  5. Open the Homebrew Channel and launch Riivolution.
  6. Once Riivolution is all loaded up, make sure "Level Patch" is enabled. Once you're ready to launch the game, select the "Launch" button to the right of the screen. Epic Mickey will then boot with your desired settings.

FAQ

  • Why isn't the patch showing up in Riivolution? - Make sure the Epic Mickey game disc is inserted and make sure the ZIP file was extracted to the root of the SD card.
  • Can this be used with Dolphin? - To load levels in Dolphin, it's recommended to use WiiScrubber or another method to directly edit the ISO file from your computer.
  • Why won't my ZIP file download? - Your browser may too old to support the patch generator. It uses relatively new web technologies, so it needs a modern browser to work properly. Try updating your browser, or downloading the latest version of Chrome, Opera or Firefox.
